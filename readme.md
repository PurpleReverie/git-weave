# Weave CLI

**Weave** is a cross-platform command-line utility (Windows, macOS, Linux) built with .NET, designed to manage sub-child git repositories — ideal for mono-repo-like workflows or nested repos.

## 🚀 Features (Planned)

- Manage nested repositories
- Synchronize child repos
- Command chaining
- Repo health checks

## 🔧 Installation

You can build from source using .NET SDK:

dotnet build
